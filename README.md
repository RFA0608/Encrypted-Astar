# 사용하기 위한 세팅
``` bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source "$HOME/.cargo/env"
cargo new tfhe_astar_test
cd tfhe_astar_test
```
를 이용하여 rust 사용 준비를 한다.

내부에 Cargo.toml을 깃허브의 것으로 교체한다.

내부 폴더 중 src의 main.rc를 깃허브의 것으로 교체한다.
